# ⚖️ Islam & Political Tyranny — الإسلام والاستبداد السياسي

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الإسلام والاستبداد السياسي (Islam and Political Tyranny)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 good governance cards with Quranic verses and lessons, 6 pillars of just governance, daily habit tracker with streaks, knowledge quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes with localStorage persistence
- Narrator with karaoke, voice/speed/pitch controls
- Responsive design + print styles
- PWA-ready with manifest.json

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.
